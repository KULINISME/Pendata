# Data Preparation

Data Preparation ialah ahapan krusial dalam metodologi data science (CRISP-DM) untuk mengenali, mengumpulkan, dan mengeksplorasi data awal guna memahami struktur, kualitas, dan potensi masalahnya (seperti missing values atau outlier). Tahap ini memastikan data relevan dengan tujuan bisnis, mencakup pengumpulan data, deskripsi, eksplorasi (EDA), dan verifikasi kualitas data sebelum pemodelan.
